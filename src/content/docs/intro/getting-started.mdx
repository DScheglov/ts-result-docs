failure, focusing on type safety, great developer experience, and an easy learning curve.

## Installation

```bash
npm install resultage
```

## What is in the Package?

- Types:
  - `Result<T, E>` = `Ok<T>` | `Err<E>`
  - `AsyncResult<T, E>` = `Promise<Result<T, E>>`
  - Utilities: `OkTypeOf`, `ErrTypeOf`, `ResultOf`
- Factory Functions:
  - `ok`, `err`
  - `asyncOk`, `asyncErr`
- Type Guards:
  - `isResult`
  - `isOk`, `isErr`
- Conditional Factory Functions:
  - `okIf`, `expect`
  - `okIfExists`, `expectExists`
- `Do`-wrapper for imperative style of chaining
- Curried versions of `Result` methods
  - `map`, `mapErr`
  - `chain`, `chainErr`
  - and more ...
- Curried methods to operate with AsyncResult
  - `thenMap`, `thenMapErr`
  - `thenChain`, `thenChainErr`
  - and more ...
- Utilities to operate with Lists of Results
  - `collect`, `collectErr`
  - `separate`
- Utilities to run `Result` and `AsyncResult` thunks:
  - `sequence`
  - `sequenceAsync`
- Functional Programming utilities:
  - `pipe`, `compose`
  - `identity` (`idX`), `lazy`, `asConst`
- Assertion utilities:
  - `assertNever`, `unreachable`
